## Project Overview

This repository contains the research project for the Master of Science in Artificial Intelligence program at the Open University of Kenya (OUK).

## Project Structure

```
mastersaiproject/
├── data/
│   └── raw/
│       ├── maize_healthy/
│       ├── maize_streak/
│       └── maize_mln/
├── notebooks/          # Jupyter notebooks for experimentation
├── src/                # Source code modules
├── models/             # Trained model files
├── docs/               # Documentation
├── requirements.txt    # Python dependencies
└── README.md
```

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/FRIEZEWANDABWA/mastersaiproject.git
   cd mastersaiproject
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
